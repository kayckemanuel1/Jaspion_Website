Bem-vindo à versão beta do projeto Jaspion

Jaspion é um software livre, desenvolvido para auxiliar microempreendedores e pequenos comerciantes a agilizar e gerenciar as suas vendas. Com funcionalidades que automatizam tarefas manuais, Jaspion surge como uma solução eficiente e acessível para melhorar a eficiência operacional e gerenciar pequenos negócios sem necessidade de investimentos financeiros. Compatível com diversas plataformas, ele é ideal para computadores com recursos limitados, permitindo a utilização em equipamentos mais antigos.

Sobre

O projeto Jaspion começou a ser desenvolvido como parte de um projeto educacional para implementar o método de Aprendizagem Baseada em Problemas (PBL) dentro do colégio CETEP João Campos. Iniciado como um simples programa de linha de comando para auxiliar operadores de caixa, ele evoluiu para uma ferramenta completa com interface gráfica e funcionalidades avançadas. E pretendemos estar sempre atualizando, colocando novas funcionalidades e melhorando ele. Nossa missão é fornecer uma solução gratuita, acessível e eficaz para pequenos negócios, ajudando-os a crescer e prosperar.
